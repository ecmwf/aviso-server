# Introduction

Aviso Server provides:

- real-time notifications over Server-Sent Events (SSE),
- historical replay from JetStream-backed storage,
- schema-driven validation and topic construction.

This documentation focuses on operational behavior and configuration, with special attention to replay and JetStream settings.

## What You Will Find Here

- End-to-end architecture and request flow.
- Deployment guidance (local vs production).
- Backend behavior and limitations.
- Full configuration reference.
- Replay/watch endpoint semantics with practical rules.

## Quick Start

1. Start with [Getting Started](./getting-started.md).
2. Choose an operating profile in [Deployment Modes](./deployment-modes.md).
3. Read [Configuration](./configuration.md) and [Configuration Reference](./configuration-reference.md).
4. Review streaming behavior in [Streaming Semantics](./streaming-semantics.md).
